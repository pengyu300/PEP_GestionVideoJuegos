### Descripción:
Proyecto de python para gestionar una colección de videojuegos utilizando diferentes estructuras de datos como listas, tuplas, conjuntos y diccionarios. 
Permite realizar operaciones CRUD, búsquedas avanzadas y obtener estadísticas sobre los videojuegos almacenados.
